Como funciona

  1. Escolha a planilha .xlsx (1ª linha = cabeçalho).
  2. Selecione o modelo .docx com placeholders.
  3. Defina a pasta base e a coluna para nome das pastas/arquivos.
  4. Gere os relatórios — e, se quiser, exporte também para PDF.

Prévia de nomes, validação de campos e cancelamento incluídos.

Placeholders prontinhos

  • ✓{COL} — texto do Excel
  • ✓{IMG:COL|w=6cm;h=4cm} — imagens por caminho/URL
  • ✓{XLS:COL|sheet=Plan1;range=A1:D20;style=Table Grid;header=1} — insere tabela
  • ✓{PDF:COL|pages=all;dpi=180;w=16cm;break=paragraph} — PDF renderizado como imagens

FAQ

Preciso saber programar?
Não. É clicar e gerar.

Como coloco imagens?
Na planilha, informe o caminho (ou file://…). Use {IMG:COL|w=6cm;h=4cm} no template.

Consigo inserir uma tabela do Excel no Word?
Sim, com {XLS:COL|…} definindo planilha, intervalo e estilo.

E PDF dentro do relatório?
Use {PDF:COL|pages=…;dpi=…}. As páginas entram como imagens.
